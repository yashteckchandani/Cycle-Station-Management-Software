## Installing Dependencies

**Mysql Connector**
`pip3 install mysql-connector-python`

**Pyfiglet**
`pip3 install pyfiglet`

**Termcolor**
`pip3 install termcolor`

## Create The Database

`mysql -u root -p`
*Enter Your Password*

`source <path to the Cycle_pad.sql>`

This will create the database needed for the app to work!
